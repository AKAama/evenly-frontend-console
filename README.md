# Evenly Frontend Console

React console app built with Vite.

## Development

```bash
npm install
npm start
```

The dev server reads `PORT` from `.env`; the current local default is
`http://localhost:3001`.

The API URL is read from:

```bash
REACT_APP_API_URL=http://localhost:8000
```

## Scripts

```bash
npm test
npm run build
npm run preview
```

Production output is written to `dist/`.
